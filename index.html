Cedura Project

Automating Employee Salary Sheets with Excel

Abstract

The Cedura Project is an Excel-based automation project designed to streamline the process of generating employee salary sheets. The project eliminates the need for manual updates by using formulas, VBA macros, and Power Query to automatically update data for new months or new employee entries. The system includes sheets for employee details, leaves, holidays, and a template that consolidates all information for salary calculation.

Data Analysis Process

Data Cleaning

Data cleaning involved removing duplicates, correcting inconsistencies, and ensuring that all employee records were up-to-date. Power Query was used to transform and clean the data before loading it into the Excel workbook.

Data Processing

Using Excel formulas like XLOOKUP, INDEX, and MATCH, the project automatically fetched employee details, attendance, and leave records. VBA macros were used to generate individual employee sheets and automate repetitive tasks.

Output

The final output was a fully automated salary sheet that could be generated for any month with minimal manual intervention. The system also provided detailed insights into employee attendance, leaves, and salary deductions.

Conclusion

The Cedura Project successfully automated the employee salary sheet generation process, reducing manual effort and minimizing errors. The project demonstrated the power of Excel, VBA, and Power Query in automating complex tasks and provided valuable insights into employee attendance and leave patterns.
