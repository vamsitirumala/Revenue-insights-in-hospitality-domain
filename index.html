Revenue Insights in Hospitality

Analyzing Revenue Trends Using Power BI, Power Query, and DAX

Abstract

This project focuses on analyzing revenue trends in the hospitality industry, including hotels, restaurants, and events. The goal is to identify key performance indicators (KPIs) such as RevPAR (Revenue Per Available Room), Occupancy Rate, ADR (Average Daily Rate), and Realization Percentage. Using Power BI, Power Query, and DAX queries, the project provides actionable insights to improve financial performance and optimize revenue generation.

Data Analysis Process

Data Collection

Data was collected from various sources, including booking systems, customer demographics, and revenue reports. The dataset included information on room bookings, cancellations, no-shows, and customer ratings.

Data Transformation

Using Power Query, the data was cleaned and transformed. This included handling missing values, removing duplicates, and creating new columns for analysis, such as week numbers and day types (weekday/weekend).

Data Modeling

In Power BI, relationships between tables were established using common columns. DAX queries were used to create calculated columns and measures, such as RevPAR, Occupancy %, and ADR.

Visualization

Interactive dashboards were created in Power BI to visualize revenue trends, booking patterns, and customer demographics. Key metrics like Realization %, Cancellation %, and No-Show Rate were highlighted.

DAX Queries and Measures

Below are the essential DAX queries used to calculate key metrics for this project:

Metric DAX Expression
Revenue SUM(fact_bookings[revenue_realized])
Occupancy % DIVIDE([Total Successful Bookings],[Total Capacity],0)
ADR DIVIDE([Revenue], [Total Bookings],0)
RevPAR DIVIDE([Revenue],[Total Capacity])

Conclusion

The project successfully identified key revenue trends and provided actionable insights for the hospitality industry. By analyzing metrics like RevPAR, Occupancy %, and ADR, the project highlighted areas for improvement and optimization. The interactive Power BI dashboards enabled stakeholders to make data-driven decisions to enhance revenue generation.

What I Gained

  • Hands-on experience with Power BI, Power Query, and DAX for data analysis and visualization.
  • Improved skills in data transformation, modeling, and creating interactive dashboards.
  • Understanding of key hospitality metrics like RevPAR, ADR, and Occupancy %.
  • Ability to derive actionable insights from complex datasets.

Struggles & Challenges

  • Handling large datasets with missing values and inconsistencies was challenging.
  • Creating complex DAX measures for metrics like Realization % required extensive testing.
  • Ensuring accurate relationships between tables in Power BI was time-consuming.
